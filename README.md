# Прогнозирование цен на недвижимость (House Prices)

Предсказание стоимости жилья на основе набора признаков из известного соревнования Kaggle **"House Prices: Advanced Regression Techniques"**. Задача регрессии, метрика качества — **RMSLE** (Root Mean Squared Logarithmic Error).

Описание задачи

Необходимо построить модель, которая по характеристикам дома (площадь, количество комнат, этаж, состояние и т.д.) предскажет его рыночную стоимость. Датасет содержит 79 объясняющих переменных (как числовых, так и категориальных) и целевую переменную `SalePrice`. Тренировочная выборка — 1460 объектов, тестовая — 1459.

Используемые библиотеки

- `pandas`, `numpy` — обработка данных
- `scikit-learn` — модели, метрики, разделение выборки

Структура проекта:
train.csv - тренировочные данные
test.csv - тестовые данные
house_price_prediction.ipynb - основной ноутбук с решением
README.md - описание проекта


Основные этапы работы:

1. Анализ и предобработка данных
- Проверка пропусков: для числовых признаков пропуски заполнены медианой, для категориальных — модой.
- Логарифмирование целевой переменной (`np.log1p`) для приведения распределения к нормальному виду.

2. Создание новых признаков (Feature Engineering)
Добавлены комбинированные признаки:
- `TotalSF` — общая площадь (подвал + 1й этаж + 2й этаж)
- `SFPerRoom` — жилая площадь на комнату
- `TotalBathrooms` — общее количество санузлов (с учётом половинных)
- `TotalPorchSF` — суммарная площадь веранд
- Бинарные признаки: наличие бассейна, гаража, подвала, камина, второго этажа
- `HouseAge`, `RemodAge` — возраст дома и после ремонта
- `OverallGrade` — произведение оценки качества и состояния
- `KitchenRatio` — отношение количества кухонь к числу комнат

3. Обработка выбросов
Удалены два аномальных объекта.

4. Кодирование категориальных переменных
Применено One-Hot Encoding (через `pd.get_dummies`). После кодирования количество признаков — 300.

5. Разделение данных
- Тренировочная выборка разбита на обучение (80%) и валидацию (20%) с `random_state=17`.

6. Моделирование
Использован градиентный бустинг (`GradientBoostingRegressor`) с параметрами:
- `n_estimators=500`
- `learning_rate=0.1`
- `max_depth=4`
- `min_samples_split=20`
- `min_samples_leaf=10`
- `subsample=0.8`

7. Оценка качества
На валидационной выборке достигнут RMSLE = 0.12599.

8. Финальное обучение и предсказание
Модель переобучена на всех тренировочных данных и применена к тестовой выборке. Предсказания сохранены в `submission.csv`.

Результаты предсказаний на тесте

- Максимальная цена: $639 416
- Средняя цена: $179 354
