# Прогнозирование цен на недвижимость (House Prices)

Описание проекта
Реализация модели машинного обучения для предсказания стоимости жилой недвижимости на основе датасета Kaggle House Prices: Advanced Regression Techniques.
Тип задачи: регрессия
Метрика качества: RMSLE (Root Mean Squared Logarithmic Error)
Цель — построить модель с хорошей обобщающей способностью на табличных данных с числовыми и категориальными признаками.
Данные
79 исходных признаков (числовые и категориальные)
1460 объектов в тренировочной выборке
1459 объектов в тестовой выборке
Целевая переменная: SalePrice
Предобработка данных
Заполнение пропусков:
числовые признаки — медиана
категориальные — мода
Логарифмирование целевой переменной (np.log1p)
→ уменьшение асимметрии распределения и стабилизация дисперсии
Удаление аномальных наблюдений
One-Hot Encoding категориальных переменных
→ итоговое число признаков: ~300
Feature Engineering
Созданы дополнительные информативные признаки:
TotalSF — суммарная площадь дома
SFPerRoom — площадь на комнату
TotalBathrooms — общее число санузлов
TotalPorchSF — суммарная площадь веранд
HouseAge, RemodAge — возраст дома и после ремонта
OverallGrade — интегральная характеристика качества
KitchenRatio — отношение кухонь к числу комнат
Бинарные признаки: наличие гаража, подвала, бассейна, камина, второго этажа
Модель
Использована модель:
Gradient Boosting Regressor
Выбор обусловлен высокой эффективностью бустинга на табличных данных и способностью моделировать нелинейные зависимости.
Параметры модели:
n_estimators = 500
learning_rate = 0.1
max_depth = 4
min_samples_split = 20
min_samples_leaf = 10
subsample = 0.8
Данные разделены на train/validation (80/20, random_state=17).
Результаты
Validation RMSLE: 0.12599
Модель демонстрирует хорошую способность обобщения и устойчивость к переобучению.
После валидации модель была переобучена на всей тренировочной выборке и использована для генерации предсказаний на тестовых данных.
